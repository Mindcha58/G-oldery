# G-oldery (Guide Old Guy)

G-oldery เป็นแอปพลิเคชันบนระบบปฏิบัติการ **Android** พัฒนาด้วย **Flutter** โดยมีวัตถุประสงค์เพื่อช่วยแนะนำแนวทางการดูแลสุขภาพของผู้สูงอายุ ลดความเสี่ยงในการเกิดโรคเรื้อรัง และส่งเสริมคุณภาพชีวิตที่ดีขึ้น

แอปพลิเคชันนี้จัดทำขึ้นเป็นส่วนหนึ่งของรายวิชา **CPE 121 Mobile Application Developments** คณะวิศวกรรมศาสตร์ สาขาวิศวกรรมคอมพิวเตอร์ มหาวิทยาลัยศรีนครินทรวิโรฒ

---

## วัตถุประสงค์ของโครงการ

* คำนวณค่าดัชนีมวลกาย (Body Mass Index: BMI) เพื่อประเมินภาวะร่างกายของผู้สูงอายุ
* แสดงปริมาณพลังงานที่เหมาะสมต่อวัน
* ให้ความรู้เกี่ยวกับโรคเรื้อรังที่พบบ่อยในผู้สูงอายุ
* แนะนำแนวทางการออกกำลังกายและการรับประทานอาหารที่เหมาะสม

---

## ฟังก์ชันหลักของแอปพลิเคชัน

### 1. การคำนวณค่า BMI

* ผู้ใช้กรอกข้อมูลน้ำหนักและส่วนสูง
* ระบบคำนวณค่า BMI และจัดหมวดหมู่ (ผอม, ปกติ, น้ำหนักเกิน, อ้วนระดับ 1–2)
* แสดงคำแนะนำด้านสุขภาพเบื้องต้น

### 2. การเลือกโรคประจำตัว

ผู้ใช้สามารถเลือกโรคประจำตัวที่เป็นอยู่ ได้แก่

* โรคอ้วน
* โรคเบาหวาน
* โรคไตเรื้อรัง
* โรคไขมันในเลือดสูง
* โรคความดันโลหิตสูง
* ไม่เป็นโรคใด ๆ

ระบบจะแสดงข้อมูลที่เกี่ยวข้อง เช่น สาเหตุของโรค วิธีการป้องกัน การออกกำลังกาย และอาหารที่เหมาะสม

### 3. แนวทางการดูแลสุขภาพ (กรณีไม่เป็นโรค)

* การออกกำลังกายที่เหมาะสมสำหรับผู้สูงอายุ
* อาหารหลัก 5 หมู่
* การป้องกันการเกิดโรค

## โครงสร้างโปรเจกต์ (โดยสังเขป)

```
lib/
 ├── main.dart          // หน้าเริ่มต้นของแอปพลิเคชัน
 ├── user.dart          // หน้ากรอกข้อมูลผู้ใช้
 ├── bmi.dart           // ฟังก์ชันคำนวณ BMI
 ├── disease/           // หน้าข้อมูลโรคต่าง ๆ
 └── guide/             // แนวทางออกกำลังกาย อาหาร และการป้องกันโรค
```
